# Statutory Audit Start Checklist

Use this when a CA wants to begin statutory-audit work inside CAClaw.

## Step 1: Confirm audit type

Make sure the work is a statutory audit, not internal audit, due diligence, forensic review, or tax audit.

## Step 2: Confirm audit stage

Identify the current stage:

- engagement acceptance
- planning
- materiality
- risk assessment
- fieldwork and execution
- completion
- reporting

## Step 3: Ask for the right output

Good requests include:

- engagement checklist
- audit strategy draft
- materiality working
- risk matrix
- audit plan
- area-specific audit program
- CARO checklist
- draft audit report
- completion memo
- management letter

## Step 4: Keep review explicit

Every output should be reviewed by a qualified CA before use in real audit documentation or reporting.
