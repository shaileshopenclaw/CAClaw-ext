# CA GST Starter Workflow

Use this skill for GST preparation and review work. Keep the output practical, reusable, and human-reviewed.

## What to do

1. Identify whether the user needs preparation, reconciliation, review, or follow-up support.
2. Read `references/gst.md` for workflow scope and usage boundaries.
3. Use `assets/gst-checklist.md` when a reusable checklist or work scaffold will help.
4. Keep the output framed as preparation and review support, not autonomous filing.

## Boundaries

- Do not imply that OpenClaw files GST returns automatically.
- Do not guarantee statutory correctness.
- Keep requests, checklists, and summaries in plain CA-friendly language.
